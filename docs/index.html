Introduction

KaneCode Studio is a web editor developed by KaneCode that allows you to easily edit any HTML code.

CSS

Copy-paste the stylesheet <link> into your <head> before all other stylesheets to load our CSS.

<link href="https://cdn.kanecode.com/libs/kanecode-studio@0.0.1/dist/css/kanecode-studio.min.css" >

JS

Place the following <script> near the end of your pages, right before the closing </body> tag, to enable it.

<script src="https://cdn.kanecode.com/libs/kanecode-studio@0.0.1/dist/js/kanecode-studio.min.js"></script>

Initialize

To use KaneCode Studio, a previous <div> element is required to initialize it. In addition, we can select some options at the time the script is executed.

<!-- HTML: Create container element -->
<div class="selector"></div>

<!-- Script: Initialize KaneCode Studio -->
<script>
	const studio = new KCStudio(document.querySelector('.selector'), {
		border: false,		// Hide the border around the studio.
	});
</script>
