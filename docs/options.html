Options

KaneCode Studio is a web editor developed by KaneCode that allows you to easily edit any HTML code.

Option Types Description
border boolean Show/Hide a border around the studio.
fullScreen boolean Enable/Disable the studio in full screen.
header object Options related to the interface header.
 - show boolean Show/Hide the header.
menus object Options of the different menus.
 - bottom object Options related to the bottom menu.
    - show boolean Show/Hide the bottom menu.
 - left object Options related to the left menu.
    - show boolean Show/Hide the left menu.
 - right object Options related to the right menu.
    - show boolean Show/Hide the right menu.
onsave function The described function takes care of saving the content of the page.

onsave

The function described in onsave takes care of saving the content of the page. This option works when the save() method is executed. The received parameter is a text with all the content.

Default function
var database = '';		// Variable where data is saved

new KCStudio(element, {
	onsave: (html) => {
		database = html;
		return true;		// Is necessary to indicate that file has been saved.
	}
});
Async function

Returns a promise to save the data asynchronously.

If the function duration is more than 30 seconds to execute, the program will understand that it could not be saved correctly and will report an error when saving. 
new KCStudio(element, {
	onsave: (html) => {
		return new Promise((resolve, reject) => {	 // Returns a value of type Promise
			var form = new FormData();
			form.append('content', html);
			fetch('save-data.php', { body: form })
				.then((res) => res.json())
				.then((data) => {
					if (data === true) {

						// Call the resolve function to indicate
						// that it has been saved successfully.
						resolve();

					} else {

						// Call the resolve function to indicate
						// that an error has occurred.
						reject();

					}
				});
		});
	}
});
