Methods

KaneCode Studio is a web editor developed by KaneCode that allows you to easily edit any HTML code.

fullscreen()

Shows/Hides the studio in full screen.

studio.fullscreen();

hideMenu(menu)

Hides the specified menu. The expected values (of type string) for the menu parameter could be bottom, left or right, depending on the menu that is expected to be hidden.

studio.hideMenu('bottom');	
studio.hideMenu('left');
studio.hideMenu('right');

open(content)

Open a new project. It receives a single input parameter (content) of type string, which is the web to load in HTML format.

var content = `
<html lang="en">
<head><title>Document</title></head>
<body>
	<h1>Hello world!</h1>
	<p>I love kanecode :D</p>
</body>
</html>`;

studio.open(content);

preview()

Hides the interface and loads the page for a real-time preview. Calling the function again will do the opposite.

studio.preview();

save()

Save the current project. In order to work, it is necessary to have previously specified the onsave option. If the project gives an error when saving, this function will throw an error message.

var studio = new KCStudio(element, {
	onsave: (html) => { database = studio }
});

studio.save();

showMenu(menu)

Shows the specified menu. The expected values (of type string) for the menu parameter could be bottom, left or right, depending on the menu that is expected to be showed.

studio.showMenu('bottom');	
studio.showMenu('left');
studio.showMenu('right');

toggleMenu(menu)

Shows/Hides the specified menu. The expected values (of type string) for the menu parameter could be bottom, left or right, depending on the menu that is expected to be showed/hidden.

studio.toggleMenu('bottom');	
studio.toggleMenu('left');
studio.toggleMenu('right');
